## Lead Tracker - Chrome Extension
Lead Tracker is a simple and efficient Chrome Extension designed to help you save and manage leads (URLs) directly from your browser.
Whether you're doing research, prospecting clients, or collecting useful links â€” this extension helps you stay organized with just one click!

## Features 
- **Save Tab Button** - Save the current tab to access it later easily.
- **Delete All Button** - Delete all saved tabs at once.
- **User-friendly interface** - Simple and intuitive UI for managing tabs quickly.
- **Persistent Storage** - Saved tabs remain available even after closing the browser.
